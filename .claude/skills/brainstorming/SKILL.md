# Brainstorming

Enforces Socratic design refinement. Use this skill to refine ideas and formulate a solid design before starting implementation.

### Run Brainstorming
```bash
python3 scripts/design.py brainstorm \
    --title "New Feature" \
    --problem "The core problem we are trying to solve" \
    --solution "The proposed solution" \
    --edge-cases "Potential pitfalls" \
    --alternatives "Alternatives considered and why rejected" \
    --doc-type f
```
